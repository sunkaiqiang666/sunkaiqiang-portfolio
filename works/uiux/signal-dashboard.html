UIUX / Dashboard

Signal Dashboard

一个面向复杂监控场景的高密度仪表盘系统，目标是在大量数据、异常状态和协作决策之间建立清晰的视觉秩序。

Role信息架构 / UI 系统 / 动效原型
OutputDesign system, key screens, prototype

设计挑战

不同角色需要在同一个界面里看到不同层级的信息：管理者看趋势，运营者看异常，执行者看任务。因此界面不能只是漂亮的数据图，而要支持快速判断。

解决策略

用三层视觉密度组织信息：实时状态、风险信号、可执行任务。通过颜色、动效和卡片优先级，让复杂内容保持可扫描。
